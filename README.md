# 🎓 Student Registration Portal

A modern, responsive, and interactive **Student Registration Portal** built using **HTML5, CSS3, Bootstrap 5, JavaScript, and jQuery**. This project allows users to explore courses, register as students, and manage student records dynamically with a clean and modern UI.
